
# Tic Tac Toe

A simple two-player Tic Tac Toe game built with HTML, CSS, and JavaScript — playable directly in the browser with no setup required.

Two players take turns marking **O** and **X** on a 3×3 grid. The game automatically detects a winner across all rows, columns, and diagonals and displays a congratulations message. Use the **Reset** button to clear the board mid-game or **New Game** to start fresh after a win.

Built using DOM manipulation, event listeners, and win-pattern logic in vanilla JavaScript.
